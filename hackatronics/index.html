EESA Hackatronics 2023

Problem Statements

These are the problem statements for the EESA Hackatronics 2023

Embedded System & IoT



Hover or click here to see the Problem Statements!



1. create a solution that can help farmers by keeping them warned about weather conditions which can affect them by accessing data from the meteorological department.
2. Develop an air quality monitoring system that tracks pollution levels and provides insights for improving public health.
3. Design a smart waste management system that uses sensors and data analytics to optimize waste collection and disposal.
4. Create a smart farming system that uses data analytics and machine learning to optimize crop yields and resource usage.
5. IoT Based Smart Waste Management System for Smart City.
6. Create a survelliance Drone using RaspberryPi. The camera use for survelliance should capture good video quality. Real-time video streaming data should be broadcasted on any preferred software platform.
7. Create a Home Automation System WITHOUT using Arduino & RaspberryPi. Create your custom PCB board that can automate a room. Automatic Light controlling and fingerprint door opening should be the main features.

Artificial Intelligence & Machine Learning



Hover or click here to see the Problem Statements!



1. Build a Machine Learning model that can predict stock prices or cryptocurrency values.
2. Create an AI system that can automatically classify and tag images or videos based on content.
3. Build an AI-powered virtual assistant that can help with scheduling, task management, and reminders.
4. Create a computer vision model that can accurately detect and classify objects in an image.
5. Build a recommendation engine for job seekers that suggests relevant job openings based on their skills and experience.
6. Select a photo album of 10 photos from which human faces can be detected, identify a person from that photos. Create a ML model that allow a program to organize photos by person. Some software like iPhoto and Google Photos has this capability.
7. Develop a Virtual Mouse using OpenCV. Create many actions using various finger combinations. This should recognise hand gestures and perform certain assigned tasks.
8. Create a Deep Iris Detection using Python. It should capture the eye blinking action.

Web & App Development



Hover or click here to see the Problem Statements!



1. Create a web-based platform that allows users to buy and sell goods and services. Firstly, you need to determine the type of goods and services that will be sold on the platform, the features and functionalities that the platform will offer. The technology stack you choose will depend on the requirements of your platform. You can select any programming language, web development framework, and database management system. The user interface is the most important part of your platform. You need to design a user-friendly interface that will allow users to easily navigate the platform, find the products or services they are looking for, and make purchases.
2. Develop a web application that helps individuals manage their finances and track expenses.
3. Create a Student Result Management System with SQL Database
4. Create a simple social media web app
5. Weather forecasting web app using ReactJS
6. Create a resume builder using HTML, CSS and ReactJS. The user interface should be easy to use. There must be multiple resume formats. The infomation given by the user must adjust automatically according to the template.
7. Clone any famous E-Commerce website using any language.
8. Create a blog portal where user can write any kind of blog, while others can like and share the blog via a unique link. Create a text editor to write the blog and add insert images option. Sort top performing blogs on the basis of popularity by likes and feature it on the homepage of the blog portal.

Robotics



Hover or click here to see the Problem Statements!



1. Develop an autonomus delivery robot that can deliver packages or other items which can be easily used in a warehouse or on a college campus. The robot should be able to navigate complex environments and interact with humans in a safe and efficient manner.
2. Create a sporty robot that can play basketball, either autonomously or with some manual inputs. The robot should be able to identify the basket accurately and put the ball in basket.
3. Develop a robot that can automate farming tasks, such as planting, harvesting, or fertilizing crops. The robot should be able to operate in a range of environments, identify and respond to changes in soil or weather conditions, and be able to learn and adapt over time.
4. Develop a robot that can assist with everyday tasks in the home or office, such as cleaning, cooking, or organizing. The robot should be able to interact with humans in a natural and intuitive way, and be easy to program and customize for different users.
5. Develop a robot that can translate any language audio into any other language.
6. Create a automation bot for Window server cyclic monthly patching bot. Input should be in csv file.
7. Create a Optimum Path finder robot that can identify the shortest path from many avaiable paths to reach the destination.

Code of Conduct

Event Guidelines:

1. Each group must consist of no more than four participants.

2. Each group may choose only one problem statement to work on.

3. In the event that any domain receives an insufficientnumber of registrations, that domain will be removed and participants will be notified.

4. Participants are expected to create original solutions to the problem statements and are strictly prohibited from using ChatGPT.

5. Plagiarism is strictly forbidden and will be considered a negative factor in the judging process.

6. Participants are allowed to use simulation software or hardware components, but they are solely responsible for ensuring that these components function properly.

7. Every group member is required to participate in the project, and this will be a factor in the judging criteria.

8. Participants will be allowed to use simulation software, opensource platforms and hardware components.

9. The judges' decision is final, and participants must respect the instructions provided.

10. Judging criteria includes novelty of the idea, complexity, clarity and details in the prescribed format, feasibility, practicality, sustainability, scale of impact, user experience and potential for future work progression.

Frequently Asked questions

What is a EESA Hackatronics?
In general a Hackathon is a social coding event that brings together computer programmers like software developers, graphic designers and user interface specialists along with industry process experts and professionals to identify issues and create software solutions within 12 hours.
Who can participate?
Everybody is welcome and can apply for Hackatronics. No matter if you are a student of VIT, VSIT or VP. You don’t have to be a programmer and you certainly don’t need to have a degree in Computer Science! Specifically, we are looking for hackers, coders, makers but also for innovators, engineers, creative people, designers and data geeks. Because our challenges are not only limited to programming!
Why should I participate in EESA Hackatronics?
Learning new skills, networking, winning cash prizes, working as a team, meet great people and having fun!
How are winners selected?
There will be an independent Jury selecting the winners with a rating system: Implementability, financial benefits, presentation skills, creativity.
How many people should a Team be?
A team should be between 3-4 people, if you don’t have a team yet you can still apply because you will have the chance to join a team on-the-spot.
What are the costs of participating?
FREE. Free for spectators. Free for participants.
I have a question or concern what should I do?
Contact us. Don’t be shy! We EESA Committee is here to help and create a great atmosphere.
I have another question that is not answered in your FAQs, what should I do?
You can directly contact us. Contact details are mentioned in the contact section.
